Función Greedy, la cual obtiene una solución utilizando una lógica localmente óptima, tomando el camino más económico.
La función Greedy crece la cantidad de vehículos necesarios de forma lineal, es decir, si no puede con 1, intenta con dos y así.
Si se utilizan 2 o más vehículos para resolver el problema, estos se turanarán para recorrer un camino, distribuyendo el riesgo.
